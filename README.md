# Eve Las Enterprise Website

A responsive, dependency-free corporate website for Eve Las Enterprise, prepared for regulatory, customs, supplier, bank, and institutional presentation.

## Included

- Company positioning and verified public-record highlights
- Aquatic, fishery, plant, floriculture, and import-coordination service sections
- Six-step shipment workflow
- Public BFAR, BPI-NPQSD, and Bureau of Customs source links
- Regulatory and privacy disclaimers
- Preliminary import-review form that opens the visitor's email application
- Responsive mobile navigation and accessible page structure
- No backend, database, environment variables, or paid dependencies

## Preview locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Deploy

Import this repository directly into Vercel and deploy with the default settings. The project is a static site, so no build command or environment variables are required.

It can also be hosted through GitHub Pages or any ordinary static web host.

## Important before formal launch or government submission

The website deliberately does not fabricate private company records. Attach and verify the originals for:

- DTI Business Name Certificate
- BIR Form 2303
- Current Mayor's Permit
- BOC accreditation, Customs Client Number, and CPRS status
- Current BFAR certificate number and expiry
- Original BPI-NPQSD certificate
- Proprietor and authorized representative details
- Current telephone numbers
- Facility and operational photographs
- Audited financial statements, ITR, and proof of financial capacity

Public facts displayed on the website were reviewed on 15 July 2026. Regulatory status must be rechecked before formal publication or filing.
